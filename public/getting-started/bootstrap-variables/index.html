Bootstrap Variables

This is a place for us to document our changes to Bootstrap source code i.e. make changes to variables. Its also where we can record colour choices per site.

Focus styling

Focus styling for interactive elements (links/buttons...) is kept consistant throughout all of our sites. We want to utilise a bright yellow background with black text / outlines, or a single yellow outline depending on the element.

Try using tab to put the two cards in focus, in order to see what the focus styling looks like.

#0b0c0c

"Focus black"

$focus-text-colour

#ffdd00

"Focus yellow"

$focus-colour
